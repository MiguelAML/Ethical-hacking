Brute-Force Attacks

A brute-force attack tries every possible password until one works.

A dictionary attack uses a list of common passwords instead.

In a pentest, a tester may use this to check if accounts have weak passwords.

Defenses

  • Account lockouts
  • Rate limiting
  • Multi-factor authentication

A finding would say: weak password allowed access. The fix is stronger passwords and login protections.
