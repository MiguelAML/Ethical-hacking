Brute-Force Attacks

A brute-force attack is when someone tries every possible password until one works. It is a method, not a single tool.

Brute Force vs Dictionary Attack

A brute-force attack tests all combinations, which can take a long time. A dictionary attack is faster because it uses a list of common passwords.

Example in a Pentest

During a penetration test, a tester might try logging into an account using many password attempts to see if weak passwords are allowed. If access is gained, it shows a serious security issue.

Defenses

  • Account lockouts after too many failed attempts
  • Rate limiting to slow down login attempts
  • Multi-factor authentication (extra verification step)

Security Finding

A tester would report that weak passwords allow unauthorized access. The recommendation would be stronger passwords and protections like MFA.
