Command Line Interface (CLI)

The CLI is a way to control a computer by typing commands instead of clicking. Unlike a graphical interface, it gives direct control over the system.

Security professionals prefer CLI because it is faster, more precise, and works remotely over networks. 

cd /etc
ls -la

This command moves into a system folder and lists all files, including hidden ones. This helps a tester quickly explore a system during an investigation.
