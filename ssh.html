SSH (Secure Shell)

SSH is a protocol that allows secure remote access to another computer over a network. It encrypts the connection so data cannot be easily intercepted.

Why SSH Is Important

Before SSH, tools like Telnet sent data in plain text, which meant usernames and passwords could be captured. SSH solves this by encrypting everything being sent.

What You Can Do With SSH

Once connected, you can control the remote system like you are sitting in front of it. This includes running commands, accessing files, and managing the system.

Example Command

ssh student@192.168.1.100

Command Breakdown

ssh = the protocol used to connect securely
student = the username on the target system
192.168.1.100 = the IP address of the target device

Real-World Example

In a lab, a tester might first use Nmap to find a device on the network, then use SSH to connect to it and investigate the system further.
