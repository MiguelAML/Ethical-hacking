SSH (Secure Shell)

SSH allows secure remote access to another computer over a network. Before SSH, tools like Telnet sent data in plain text, which was unsafe.

Once connected, you can control the remote system like you are sitting in front of it. 

ssh student@192.168.1.100

student = username
192.168.1.100 = target IP address
ssh = protocol used to connect securely
