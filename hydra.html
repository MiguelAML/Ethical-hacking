Hydra

Hydra is a tool used to test usernames and passwords automatically.

A wordlist is a file with many possible passwords. Hydra tries each one. 

hydra -l admin -P rockyou.txt ssh://192.168.1.50

-l admin = username
-P = password list
ssh:// = protocol and target

Ethics

Running Hydra without permission is illegal because it is an attack. Ethical hackers only use it in authorized tests.

Defenses include account lockouts, rate limiting, and multi-factor authentication.
