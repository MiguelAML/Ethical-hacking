Nmap (Network Mapper)

Nmap is a tool used to scan networks and discover devices and services. Network scanning means checking what systems are online and what they are running.

What Are Ports?

Ports are like doors on a computer. Each open port shows a service running, such as a website or remote login. Knowing which ports are open helps a security tester find possible entry points.

Example Command

nmap -sV 192.168.1.0/24

-sV = detects service versions
192.168.1.0/24 = scans all devices on the network

Sample Output

22/tcp   open   ssh
80/tcp   open   http
3306/tcp open   mysql

What the Results Mean

Port 22 shows SSH is running, which allows remote access. Port 80 shows a web server. Port 3306 means a database is exposed on the network. A tester would investigate these services to check for weaknesses.
