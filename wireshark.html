Wireshark

Wireshark is a tool used to capture and analyze network traffic in real time. It allows security testers to see what data is being sent between devices.

What is a Packet?

A packet is a small piece of data sent over a network. You can think of it like an envelope that carries information from one computer to another.

What Wireshark Shows

Wireshark captures packets and breaks them down so you can see details like source, destination, and the type of data being sent.

Security Example

If a website uses HTTP instead of HTTPS, login data is not encrypted. Wireshark can reveal usernames and passwords in plain text, which is a major security risk.

Why It Matters in Cybersecurity

Security testers use Wireshark to detect unsafe traffic, misconfigured services, and potential data leaks on a network.
